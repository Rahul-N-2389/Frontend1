VLSI ARCHITECTURES FOR SECURITY ANALYSIS WITH DUAL- KEY LFSR USING BARREL SHIFTER AND S-BOX

  • The increasing demand for secure and efficient encryption methods in hardware implementations has led to the development of optimized VLSI architectures.
  • This project presents a novel security framework incorporating a Dual-Key LFSR with a Barrel Shifter and an S-Box for enhanced cryptographic performance.

  • The S-Box adds non-linearity and confusion, strengthening the cryptographic robustness of the system.
